# 🎬 CinePulse

CinePulse is a sleek, modern, and lightning-fast movie discovery web application built with **React** and **Vite**. It features real-time search filtering, dynamic movie grids, and a clean, fully responsive user interface designed for film enthusiasts to explore their favorite cinema effortlessly.

---

## ⚡ Features

- **Instant Search:** Real-time client-side movie filtering that updates the movie grid dynamically as you type.
- **Sleek UI/UX:** A beautiful, responsive card layout tailored perfectly for mobile, tablet, and desktop viewports.
- **Modern React Architecture:** Built using functional components, state hooks (`useState`), and clean component state-lifting patterns.
- **Environment Security:** Secured API integrations using `.env` files to prevent secret key exposure on public repositories.
- **Ultra-Fast Performance:** Powered by Vite for optimized production bundles and blazing-fast development reload times.

---

## 🛠️ Tech Stack

- **Frontend Library:** React (Functional Components & Hooks)
- **Build Tool:** Vite (ES Next-generation frontend tooling)
- **Styling:** CSS3 / Tailwind CSS (Fully Responsive layout)
- **State Management:** React Component Props & Lifting State
